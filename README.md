Project Overview
A comprehensive Power BI dashboard designed to analyze sales performance, financial health, and operational efficiency for a retail business operating in Saudi Arabia. The project transforms raw transactional data into actionable strategic insights.

Key Features
Executive Summary: A high-level overview of KPIs, including Actual Sales ($18.38M), Target Achievement (86%), and YoY Growth.

Geographic Analysis: Interactive map showing sales distribution across key cities in KSA (Riyadh, Jeddah, Dammam, etc.).

Product Performance Deep-Dive: Analysis of product categories using Scatter Charts with a Play Axis to visualize performance trends over time.

Financial Insights: Profit margin analysis and order fulfillment status tracking.

Technical Highlights (The "Magic" Behind the Scenes)
Advanced DAX: * Created Dynamic Tooltips using DAX measures to provide automated text summaries (e.g., "In [City], we achieved [Sales] which is [X]% of the goal").

Time Intelligence functions for Year-over-Year (YoY) comparisons.

UI/UX Design: * Developed a custom Navigation System and a professional Cover Page for an intuitive user experience.

Used a consistent color palette (Purple/White) to ensure visual hierarchy and readability.

Data Modeling: Established a robust Star Schema to link sales, targets, and calendar tables efficiently.

Tools Used
Power BI Desktop (Visualization & Modeling)

DAX (Business Logic & Calculations)

Power Query (Data Cleaning & Transformation)

How to Use
Open the .pbix file in Power BI Desktop.

Start from the Project Cover page and click "Click to Start".

Use the top navigation bar to switch between analysis views.

Hover over data points to see the Smart Tooltips in action.
